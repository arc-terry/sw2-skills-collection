# Template Skill

Use this skill as a starting point when creating new skills in this repository.

## When to use

- You are authoring a new skill and need a compliant structure.
- You want a consistent style for instructions and supporting files.

## Steps

1. Copy this folder and rename it to your target skill name.
2. Update frontmatter values (`name`, `description`, optional fields).
3. Replace instruction sections with task-specific guidance.
4. Add supporting files only when they provide clear value.

## Supporting resources

- Detailed guidance: `references/REFERENCE.md`
- Example helper script: `scripts/example.sh`
